
STM32家庭环境监测系统（机智云）功能如下：

1）检测功能：环境温湿度数据、烟雾浓度数据、PM2.5浓度数据。

2）显示功能：将检测到的环境数据显示在oled屏幕上。

3）控制功能：系统可以控制风扇、加湿器以及步进电机。

4）手动模式：通过手机APP控制风扇、加湿器和步进电机的运行。

5）自动模式：当环境温度超过阈值打开风扇、当环境湿度低于阈值打开加湿器，环境烟雾浓度高于阈值打开风扇并报警、环境PM2.5浓度超过阈值打开窗户并报警。
